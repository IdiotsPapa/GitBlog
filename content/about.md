## 소개

C# / ASP.NET MVC4부터 Core 3.1, .NET 8까지 사내 웹·ERP성 시스템을 개발하고 운영해 왔습니다.

출장·교육 정산처럼 규칙이 많은 업무를 화면과 데이터로 옮기는 일을 해 왔고,  
최근에는 Cursor·Claude 등 AI 코딩 도구로 React·Flutter·자동화 파이프라인을 실험하고 있습니다.
AI가 만든 결과물은 직접 검증하고, 사내 코드와 데이터는 외부에 올리지 않습니다.

## 주로 다루는 것

- 사내 웹 / ERP·결재 연계 / 운영·모니터링
- 레거시 분석과 점진적 고도화
- AI 코딩 도구 활용과 검증 과정
- Flutter / React 등 사이드 실험
