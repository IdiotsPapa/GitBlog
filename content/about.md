## 소개

C# / ASP.NET **MVC4 → Core 3.1 → .NET 8** 환경에서 사내 웹·ERP성 시스템을 개발·운영하는 개발자입니다.

출장·교육 정산처럼 규칙이 많은 업무를 화면과 데이터로 옮기는 일을 해 왔고,  
최근에는 Cursor·Claude 등 AI 코딩 도구로 React·Flutter·자동화 파이프라인을 실험합니다.  
AI는 속도를 위해 쓰되 **결과물을 검증**하고, **사내 코드·데이터는 외부에 올리지 않는** 쪽을 기준으로 삼습니다.

## 주로 다루는 것

- 사내 웹 / ERP·결재 연계 / 운영·모니터링
- 레거시 분석과 점진적 고도화
- AI 코딩 도구를 활용한 생산성·검증 습관
- Flutter / React 등 사이드 실험
