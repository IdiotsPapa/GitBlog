사내 제안관리 시스템

2019 — 운영개발·운영work
ASP.NET MVC Oracle C#

임직원 개선 아이디어를 등록하고, 검토·실시·등급 심사·확정·포상까지 관리하는 사내 제안관리 시스템을 구축했습니다. 아이디어를 모아 두는 게시판이 아니라, 평가와 보상·실적 집계까지 이어지는 업무 시스템입니다.

  • 기술: C# / ASP.NET MVC, Oracle
  • 상태: 기능 개발 후 운영·유지보수

구성한 기능

제안자가 개선 전·후 내용과 추진자를 정해 등록하면, 검토·승인·반려를 거친 뒤 실시 결과를 남깁니다. 심사 단계에서는 항목별 점수를 합산해 등급을 매기고, 주관팀이 최종 확정하거나 재심사를 요청할 수 있습니다.

포상은 제안자·추진자 등 여러 수혜자에게 분배율을 나눠 반영하고, 지급·전송 상태를 관리합니다. 개인·팀·부문별 목표와 실적, 포상 현황은 대시보드와 Excel 리포트로 확인합니다. 메일·웹 알림과 미결재 건 처리도 운영에 포함했습니다.

작업에서 신경 쓴 점

등록만 되고 끝나지 않도록, 실시·심사·포상이 같은 건 안에서 이어지게 화면을 나눴습니다. 분배율은 합이 맞아야 지급이 가능하도록 검증했고, 등급·포상 기준은 관리자가 바꿀 수 있게 기준정보로 분리했습니다.

내부 조직 코드·계정·DB 식별자는 보안상 공개하지 않습니다.
