사내 제안관리 시스템

2019 — 운영개발·운영work
ASP.NET MVC Oracle C#

임직원의 개선 아이디어를 등록하고, 검토·실시·등급 심사·확정·포상까지 관리하는 사내 제안관리 시스템을 구축했습니다. 임직원이 다양한 제안을 낼 수 있도록, 제안에 대한 평가부터 보상까지 이어지게 만든 시스템입니다.

  • 기술: C# / ASP.NET MVC, Oracle
  • 상태: 개발 후 운영·유지보수

주 흐름: 임시 저장 → 검토 대기 → 실시 완료서 → 등급 부여 → 최종 확정 → 완료

구성한 기능

제안자가 개선 전·후 내용과 추진자를 정해 등록하면, 검토·승인·반려 후 실시 결과를 남깁니다. 심사에서는 항목별 점수를 합산해 등급을 매기고, 주관팀이 최종 확정하거나 재심사를 요청할 수 있습니다.

포상은 제안자·추진자 등 여러 수혜자에게 분배율을 나눠 반영하고 지급·전송 상태를 관리합니다. 개인·팀·부문별 목표·실적·포상은 대시보드와 Excel 리포트로 확인합니다. 메일·웹 알림과 미결재 건 처리도 포함합니다.

작업에서 신경 쓴 점

통합 시스템이다 보니 계열사마다 등급·심사 기준이 달랐습니다. 그래서 회사별로 기준을 맞춰 적용할 수 있게 구성했습니다.

한 건의 제안이라도 제안자가 여러 명일 수 있고, 제안만 한 사람뿐 아니라 적용·추진에 함께한 사람도 보상을 받을 수 있어야 했습니다. 제안자·추진자 등에게 분배율을 나눠 주고하도록 했고, 합이 100%가 아니면 지급되지 않게 검증했습니다. 등록만 되고 끝나지 않도록 실시·심사·포상도 같은 건 안에서 이어지게 화면을 나눴습니다.
