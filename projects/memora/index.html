Memora — AI 학습 카드 앱

2026.05 — 07개발mobile
React Native TypeScript Expo Claude

필기 사진·PDF·텍스트를 플래시카드·퀴즈로 바꾸고 복습까지 이어 주는 모바일 앱입니다.

  • 작업 기간: 2026년 5월 15일 ~ 7월 10일
  • 상태: 핵심 학습 흐름 베타 (일부 부가 기능은 비활성)
  • 플랫폼: Expo / React Native / TypeScript

카드를 만드는 것보다 어려웠던 일

Claude로 카드를 만드는 것보다, “답이 원본 어디 근거인가”가 더 중요했습니다. 카드에 원본 이미지 인덱스·위치 정보를 저장하고, 학습 중 노트로 바로 돌아갈 수 있게 했습니다.

학습 자료 등록 → AI 카드·퀴즈 생성 → 피드·퀴즈 학습
  → 결과·약점 확인 → 다음 복습 일정 계산

자료 입력과 저장

사진·PDF·텍스트를 등록할 수 있습니다. PDF를 JSON base64로 보내면 용량이 커져, Claude Files API로 먼저 업로드하도록 바꿨습니다. 생성된 덱·카드·퀴즈는 트랜잭션으로 한꺼번에 기록하고, Android SQLite에 undefined가 들어가지 않도록 값을 정규화했습니다.

학습 자료·덱·카드·퀴즈·로그·계획은 Expo SQLite에 두고, AI가 필요할 때만 외부 API를 호출합니다. 카드는 알고 있음/모름, 퀴즈는 객관식·빈칸·O/X·직접 입력을 지원하며 SM-2로 복습 일정을 계산합니다.

만들었지만 켜지 않은 기능

Three.js 3D 학습 방도 구현했으나 안정화가 부족해 기능 플래그로 베타에서 제외했습니다.

남은 검증

E2E·스토어 배포는 남아 있고, 클라이언트에 AI API 키를 두는 구조는 프로덕션에 적합하지 않습니다. 정식 출시 제품이 아니라 핵심 학습 경험을 검증한 후기 MVP입니다.
